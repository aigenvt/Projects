Introduction

Python is a high-level, interpreted, general-purpose programming language known for its readability and simplicity. It was created by Guido van Rossum and first released in 1991. Python emphasizes code readability and uses indentation (whitespace) to define code blocks instead of braces {}.

Widely used in both academia and industry. Huge community and tons of libraries. Great language for both beginners and professionals.

Simple and Readable – Python’s syntax is close to plain English, making it beginner-friendly. Interpreted – No need to compile code; you can run it directly using the Python interpreter. Dynamically Typed – You don’t have to declare variable types explicitly. Object-Oriented – Supports classes, inheritance, and encapsulation. Extensive Standard Library – Comes with built-in modules for math, file handling, networking, etc. Cross-Platform – Works on Windows, macOS, and Linux.

  • Data Science & Machine Learning (using libraries like Pandas, NumPy, TensorFlow)

    Hello World

    Here’s a simple Python “Hello, World!” example:print() is a built-in Python function that outputs text to the console. Anything inside the quotation marks (" " or ' ') is printed exactly as written.

    print("Hello, World!")

    If you save this in a file named hello.py and run it with:

    python hello.py

    you’ll see:

    Hello, World!
    Variables

    In Python, variables are used to store data values — like numbers, text, or lists — so you can reuse them later.

    name = "Alice"
    age = 25
    height = 1.68
    is_student = True
    print(name)
    print(age)
    print(height)
    print(is_student)

    Outputs:

    Alice
    25
    1.68
    True
  • Must start with a letter or underscore (_)
  • Can’t start with a number
  • Can only contain letters, numbers, and underscores
  • Are case-sensitive
  • PS:Use meaningful names for variables

    • ✅ Valid: age, user_name, _count

    ❌ Invalid: 2cool, user-name, my var
